# BasicBtn

## Demo

```html
<basic-btn
  normal="skyblue"
  hover="deepskyblue"
  active="lightskyblue"
  label="blue"
  path="#blue"
></basic-btn>
```

## Attribute

| summary             | attribute | value | default | type        |
| ------------------- | --------- | ----- | ------- | ----------- |
| button normal color | normal    |       | empty   | color       |
| button text         | label     |       | empty   | string      |
| button path         | path      |       | empty   | path or URL |

## CSS Variable

| variable name      | type | default |
| ------------------ | ---- | ------- |
| --normal-font-size | px   | 12px    |
